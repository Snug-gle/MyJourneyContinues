### Object
#### 왜 모든 클래스는 Object 클래스의 상속을 받을까?
- Object 클래스에 있는 메소들을 통해서 클래스의 기본적인 행동을 정의할 수 있기 때문.
#### Object 클래스에 선언되어 있는 메소드는 객체를 처리하기 위한 메소드와 쓰레드를 위한 메소드로 나뉜다.
- 객체 처리 관련 
	- clone, equals, finalize, Class < ? > getClass, hashCode, toString
